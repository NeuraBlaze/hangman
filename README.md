# Akasztofa

Mobilbarat klasszikus Hangman/Akasztofa jatek GitHub Pages-hez.

## Funkciok

- Magyar ekezetes betuk
- Vegyes, allatok, varosok es targyak kategoriak
- Rajzolt akasztofa 6 hibalehetoseggel
- Billentyuzet es kepernyos betugombok
- Helyi statisztika es sorozatmentes `localStorage` alapon

## Inditas

Nyisd meg az `index.html` fajlt bongeszoben.
